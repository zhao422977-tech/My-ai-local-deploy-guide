# My-ai-local-deploy-guide
广东省高二学生本地AI部署的避坑指南

---

## 一、硬件（我花钱最少的一步）
### 1.1 电脑
- 显卡挑3060以上的买显存必须8G以上
- CPU这玩意AI好像并不是很挑
- 内存条16+越多对自己的头发越好

### 1.2 软件（你如果是第一次配置很折腾）
- Python不要用最新的但也不要太老3.xxx多的就行
- Ollama你得有这玩意是跑本地的模型有用的
- 另外自备点学术VPN或者找点下三滥VPN这些能让你有更多选择
- 这个不会不要用豆包这个你得沉下心慢慢搞
- 实在是第一次不想搞去八戒那找个20块的叫他帮你搞定
- 你如果装了360类开头的玩意（无论壁纸，浏览器）要么重装系统要么找大佬教你咋删干净
- if你装过360问豆包咋重置环境，（他们多半是你报错的原因）

---

## 二、模型（最省心的 放他下载等睡醒基本都好了）
### 2.1 模型下载
- 选择适合本地部署的轻量模型
- 你在国内就跑魔塔就行别去国外网站折腾
- 模型报错找豆包叫他生成bat文件找个记事本粘贴编码选ansi另存就行
- Trae CN很好用直接让他去指定地方下载
- 实在懒得动八戒找个20块的

### 2.2 运行
- Ollama/Llama等工具快速上手
- 用ollama的时候别用它的线上下载去魔塔上下载后听豆包的
- 启动报错排查（通常是端口问题问问豆包）
- 豆包修完也可能报错但是报错也会少很多一部份剩下的叫Trae CN搞
- 如果你是装open claw那类的要网关令牌（trae cn很好用直接问他然后给权限他会直接告诉你）
- 别忘了时间运行是一个检验你成果的过程

---

## 三、焚决
### 3.1 Trae CN
- 这玩意是真正意义上的双刃剑你得知道但是他真的能帮什么语言也不懂的你搞定绝大部分的的问题
- 每当他提出方案并作出行动之后把他的日志发给deepseek小心驶得万年船

### 3.2 豆包的讨价还价
- 在与豆包相处的过程中多拍照沟通少语言交流（你的日常行为语言可能是某某专有名词）
- 当要修改文件而你却不知道文件在哪里时叫他生成BAT一键式文本（虽然可能经常报错但比你傻傻找强）
- 豆包说最终，绝对....等让你试图相信他的话时你就把他的东西发给deepseek并把deepseek的东西发给他
- 豆包生成的文件如果报错就改。别说咋样咋样谁的文件不报错？谁的文件没毛病？他们已经很好了
